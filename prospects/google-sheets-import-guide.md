# Google Sheets Import

1. Open Google Sheets on desktop or mobile.
2. Create a blank sheet named `Dentist Prospect Tracker`.
3. Import `dentist-prospects.csv`.
4. Manually review each linked Google Maps search and replace candidate rows with verified clinics.
5. Do not contact a prospect until the public facts and website problems are verified.

The initial rows are research starting points, not completed prospect audits.
